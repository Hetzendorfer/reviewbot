# TM-07: Testing — Endpoint and Component Tests

## Description
Write tests for the new stats and review history API endpoints. Ensure correct aggregation, pagination, authentication, and edge cases.

## Affected Files
- **New**: `tests/stats.test.ts` — tests for the stats endpoint logic
- **New**: `tests/reviews-api.test.ts` — tests for the review history endpoint logic

## Test Cases

### Stats Endpoint (`TM-01`)
1. **Correct totals** — seed DB with known reviews, verify `SUM` of tokens matches
2. **Date range filtering** — reviews outside the range are excluded
3. **Provider breakdown** — multiple providers/models grouped correctly
4. **Daily breakdown** — correct daily buckets
5. **Empty installation** — returns zero totals, empty arrays
6. **Failed reviews excluded** — reviews with `status='failed'` don't contribute to token sums
7. **Auth required** — returns 401 without session

### Review History Endpoint (`TM-03`)
1. **Basic pagination** — page 1 returns first 20, page 2 returns next batch
2. **Pagination metadata** — `total`, `totalPages` are correct
3. **Status filter** — filtering by `completed` excludes `failed` reviews
4. **Ordering** — most recent reviews come first
5. **Limit cap** — requesting `limit=999` is capped at 100
6. **Empty result** — returns empty array with `total: 0`
7. **Auth required** — returns 401 without session

### Metrics Extension (`TM-02`)
1. **Token totals present** — `/metrics` response includes `tokens` object
2. **Backward compatible** — existing `reviews` and `queue` fields still present

## Implementation Notes
- Use Bun's built-in test framework (`bun:test`)
- For DB tests, consider using a test database or mocking the Drizzle queries
- Follow existing test patterns in `tests/crypto.test.ts` and `tests/parser.test.ts`

## Acceptance Criteria
- [ ] All test cases above pass
- [ ] Tests can run with `bun test tests/stats.test.ts` and `bun test tests/reviews-api.test.ts`
- [ ] No flaky tests — deterministic seed data

## Depends On
- TM-01, TM-02, TM-03 (endpoints must exist)
